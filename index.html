Los 5 pilares reales de defensa forense

```
01
🔑 Código fuerte alfanumérico
El teléfono guarda todo cifrado, como en una caja fuerte. El código es la llave. Si es corto o solo números, una máquina puede probarlo miles de veces por segundo hasta adivinar. Con una contraseña larga con letras y números, eso se vuelve prácticamente imposible.
Cómo hacerlo Ajustes → Face ID y código → Cambiar código → Opciones → Código alfanumérico personalizado → Elegir algo de 10+ caracteres. Ej: miPerro2019!
02
☁️ ADP activado (Protección Avanzada iCloud)
Todo lo que guardás en iCloud (fotos, mensajes, notas) normalmente Apple lo puede ver y entregar si alguien lo pide. Con esta opción activada, Apple ya no puede ver nada — solo vos tenés la llave. Es el cambio más importante que podés hacer.
Cómo hacerlo Ajustes → [tu nombre] → iCloud → Protección de datos avanzada → Activar. Te va a pedir configurar un método de recuperación (guardalo en papel en un lugar seguro).
03
⚡ Apagar el teléfono ante cualquier riesgo
Cuando el teléfono está prendido y desbloqueado, la información está "abierta" en la memoria. Las herramientas forenses aprovechan ese momento. Si lo apagás completamente, esa información desaparece de la memoria y todo queda cifrado de nuevo. Bloquear la pantalla NO es lo mismo que apagar.
Cómo hacerlo Presioná el botón lateral + cualquier botón de volumen al mismo tiempo → aparece la pantalla → deslizá "Apagar". También podés activar Modo Avión primero para cortar toda conexión y después apagar.
04
💬 Signal con mensajes que se autodestruyen
WhatsApp guarda copias de tus chats. Telegram guarda todo en la nube. Signal es diferente: cifra todo de extremo a extremo y, con la opción de desaparición activada, los mensajes se borran solos después de un tiempo. Si alguien agarra el teléfono, no hay historial que ver.
Cómo hacerlo Instalá Signal desde el App Store. Dentro de cada chat → tocá el nombre arriba → Mensajes temporales → elegí 1 semana (o menos). Para activarlo globalmente: Signal → Ajustes → Privacidad → Mensajes temporales por defecto.
05
🔟 Borrado automático tras 10 intentos fallidos
Si alguien intenta adivinar tu código a la fuerza (probando combinaciones), al décimo intento el teléfono se borra solo. Esto hace que los ataques automáticos de prueba y error sean inútiles. No hay que tener miedo de activarlo: el teléfono te avisa antes del intento 10.
Cómo hacerlo Ajustes → Face ID y código → bajar hasta el final → Borrar datos → activar. Listo. No afecta el uso normal del teléfono.
✓ CHECKLIST COMPLETADO — DISPOSITIVO ENDURECIDO
```
